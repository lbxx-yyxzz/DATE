# 中国全年龄段抑郁风险诱因分析 (CFPS 2022)

## 1. 项目简介
本项目旨在利用机器学习算法探讨中国居民抑郁风险的社会学诱因。
资料来源：通过查阅 《中国精神卫生调查》 等权威学术报告确立研究框架。
数据获取：实证数据取自 中国家庭追踪调查 (CFPS) 2022 个人库（样本量 N=27,001）。
变量提取：编写关键词检索脚本，在数千个变量中精准定位 CES-D 抑郁量表（qq401 系列）、个人收入（incomea）及上网时长（qp601）等关键指标。

## 3.  核心研究过程
项目分为三个核心技术阶段：
数据清洗与标准化：剔除无效非法值，针对 2022 版变量特性采用中位数填充策略，最大限度保留全量样本的特征代表性。
探索性数据分析 (EDA)：通过箱线图证实抑郁风险随年龄呈“阶梯式”增长，并发现青少年群体存在严重的两极分化现象。
随机森林建模：构建分类模型量化诱因，发现年龄 (Age) 权重高达 0.51，成为首要预测因子。

## 4. 核心研究发现
- **EDA 洞察**：抑郁风险随年龄呈“阶梯式”增长，老年组重心显著偏高。
- **模型表现**：随机森林模型在 2.7 万样本上表现稳健，年龄权重 0.51，收入权重 0.38。
