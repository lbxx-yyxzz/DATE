# 中国全年龄段抑郁风险诱因分析 (CFPS 2022)

## 1. 项目简介
本项目基于 CFPS 2022 个人问卷数据（N=27,001），利用随机森林机器学习算法对全年龄段人群的抑郁风险进行实证画像。研究发现年龄（Age）是抑郁风险的首要诱因（权重 0.51）。

## 2. 项目结构
- `analysis.ipynb`: 核心代码，包含数据清洗、EDA（探索性分析）及模型训练。
- `data/`: 存放原始 DTA 数据（需自行从官网下载）。
- `ppt/`: 包含项目汇报 PPT。
- `README.md`: 项目运行说明。

## 3. 运行环境
- Python 3.13+
- 核心依赖库：pandas, numpy, seaborn, matplotlib, sklearn

## 4. 核心研究发现
- **EDA 洞察**：抑郁风险随年龄呈“阶梯式”增长，老年组重心显著偏高。
- **模型表现**：随机森林模型在 2.7 万样本上表现稳健，年龄权重 0.51，收入权重 0.38。
